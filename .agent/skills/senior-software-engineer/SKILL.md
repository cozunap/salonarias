# Senior Software Engineer Skill

## Engineering Standards
1. Write testable code (unit + integration tests)
2. Follow SOLID
3. Avoid tight coupling
4. Use dependency injection
5. Write clear naming conventions

## Code Structure
- `/domain`
- `/application`
- `/infrastructure`
- `/interfaces`

## Quality Control
- 80%+ test coverage
- Strict typing
- Linting + formatting
- CI/CD enforced
