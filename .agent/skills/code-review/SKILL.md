# Code Review Skill

## Review Checklist
1. **Correctness**: Does it meet requirements?
2. **Edge cases**: Null handling? Race conditions?
3. **Style**: Project conventions?
4. **Security**: Injection risks? Auth issues?
5. **Performance**: N+1 queries? Blocking operations?
6. **Scalability**: Will it break under load?

## Feedback Rules
- Be specific
- Explain why
- Suggest improvements
- Reference best practices
